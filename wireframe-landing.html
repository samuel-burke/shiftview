Features

Everything your team needs

One dashboard for managers and employees — no extra tools required.

🗓

Visual weekly schedule

Drag-and-drop shifts on a clean grid. Set recurring patterns, copy weeks, and publish in one click.

🟢

Live attendance

See who's clocked in right now. Employees tap to clock in from any device — no hardware needed.

🔔

Push notifications

Schedule changes and shift reminders arrive instantly on iOS and Android, even when the app is closed.

📋

Availability tracking

Employees set their typical availability. Managers see it inline while building the schedule.

📊

Coverage thresholds

Define minimum headcount per day. The dashboard warns you before you're understaffed.

🔒

Role-based access

Managers see everything. Employees see their own schedule and nothing else — no config required.

How it works

Up and running in minutes

1

2

3
